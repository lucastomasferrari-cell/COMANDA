System Requirements ​

This page lists the full system requirements for running Forkiva Restaurant POS System.
Forkiva is a modular application with a Laravel backend and a Vue 3 + Vuetify frontend, so both backend and frontend environments need to be configured properly.

Backend (Laravel) ​

Server Requirements ​

  • PHP: ^8.3
  • Web Server: Apache 2.4+ or Nginx 1.20+
  • Database: MySQL 8.0+ (recommended) or MariaDB 10.6+
  • Composer: ^2.5
  • Node.js & npm/yarn (for asset building, see frontend section)
  • Browsershot (PDF/Screenshots):
    • Node.js: >=22.0 (LTS or higher)
    • Puppeteer (Node library): >=23.0
  • Extensions (must be enabled in php.ini):
    • ctype
    • curl
    • dom
    • fileinfo
    • filter
    • hash
    • mbstring
    • openssl
    • pcre
    • pdo
    • session
    • tokenizer
    • xml
    • intl
  • Cache Drivers (one of the following):
    • Redis
    • Predis (via cache/predis-adapter)
    • Filesystem Cache

Laravel Dependencies ​

Key backend dependencies from composer.json:

  • laravel/framework:^12.0
  • nwidart/laravel-modules:^12.0 (modular architecture)
  • maatwebsite/excel:^3.1 (Excel export/import)
  • staudenmeir/belongs-to-through:^2.17 (advanced relationships)
  • swayok/alternative-laravel-cache:^6.1
  • php-http/guzzle7-adapter:^1.1
  • Development:
    • barryvdh/laravel-ide-helper
    • laravel/pint
    • laravel/sail
    • phpunit/phpunit:^11.5

Frontend (Vue 3 + Vuetify) ​

Node.js Environment ​

  • Node.js: ^22.x (LTS recommended)
  • Package Manager: Yarn 1.22.x (project uses Yarn by default)

Vue.js Dependencies ​

Key frontend dependencies from package.json:

  • Core

    • vue:^3.5
    • vuetify:^3.9.5
    • pinia:^3.0.1
    • vue-router:^4.5.0
    • vite:^6.2.2

  • UI & Components

    • @fontsource/cairo
    • vue-toastification:^2.0.0-rc.5
    • vue-tree-dnd:^0.2.4
    • vuedraggable:^4.1.0
    • vue3-perfect-scrollbar:^2.0.0
    • qrcode-vue3:^1.7.1

  • Utilities

    • axios:^1.10.0
    • lodash:^4.17.21
    • date-fns:^4.1.0
    • @vueuse/core
    • @vueuse/math

  • Charts

    • chart.js:^4.5.0

  • Icons

    • @iconify/vue
    • @iconify-json/tabler
    • @iconify-json/mdi
    • @iconify-json/ic
    • @iconify-json/bx

  • Build & Tooling

    • unplugin-auto-import
    • unplugin-vue-components
    • unplugin-vue-router
    • vite-plugin-vuetify
    • vite-plugin-vue-layouts-next
    • vite-svg-loader

Development Tools ​

  • TypeScript: ~5.8.2
  • Vue TSC: ^2.2.8
  • ESLint: ^9.23.0 (with Vuetify ESLint config)
  • Sass (Dart Sass Embedded): ^1.86.3

Recommended Setup ​

  • Local Development:

    • PHP 8.3 installed locally (or via Docker/Laravel Sail)
    • Node.js 22 LTS + Yarn
    • MySQL 8.x
    • Redis (optional, for caching and queues)
    • Supervisor (for queue workers in production)

  • Production Deployment:

    • Linux server (Ubuntu 22.04 LTS or similar)
    • Nginx + PHP-FPM
    • MySQL/Redis
    • SSL (Let’s Encrypt or custom)
    • CI/CD pipeline for automated builds

Quick Checklist ​

✅ PHP 8.3+ with required extensions
✅ MySQL 8.0+ or MariaDB 10.6+
✅ Composer 2.5+
✅ Node.js 22.x + Yarn 1.22.x
✅ Redis or Filesystem cache
✅ Proper .env configuration for backend & frontend
✅ Permissions configured for storage/ and bootstrap/cache/

This requirements page ensures your system is ready to run Forkiva smoothly in both development and production environments.
