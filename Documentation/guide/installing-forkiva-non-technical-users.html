Installing Forkiva for non-technical users ​

Welcome to Forkiva Restaurant POS 👋
This guide explains how to install Forkiva on shared hosting (cPanel, Plesk, DirectAdmin, etc.) without using the terminal.
All steps are done using your hosting control panel or File Manager.

Step 1 — Upload Files ​

  1. Log in to your hosting control panel (e.g. cPanel).

  2. Open File Manager.

  3. Upload the file:

    Forkiva/Forkiva.zip

    to the root directory of your domain:

    public_html/

Step 2 — Extract Files ​

  1. After uploading, right-click Forkiva.zip → Extract.
  2. Make sure the contents of Forkiva are placed directly inside:
    public_html/
    ⚠️ Do not keep them in a subfolder like public_html/forkiva.

Step 3 — Create a Database ​

  1. In your hosting control panel, go to MySQL Databases.
  2. Create a new database (e.g. forkiva_db).
  3. Create a new database user and password.
  4. Assign the user to the database with All Privileges.

Keep these details handy:

  • Database Name
  • Database User
  • Database Password
  • Host (usually localhost)

Step 4 — Run the Installer ​

  1. Open your browser and visit:
    https://yourdomain.com/installer
  2. The Forkiva Installer will appear.
  3. Click Start Installation.

Step 5 — Follow Installation Wizard ​

The installer will guide you through:

  1. You’ll see a Welcome screen.
    Click Start Installation.

  1. System Requirements
  • The installer will check your computer/server.
  • If everything shows OK, click Next.
  • If something is not OK, ask your hosting provider or IT support to enable it.

  1. Permissions
  • The installer checks if the system can write to important files.
  • If all show Writable, click Next.
  • If not, you may need to adjust folder permissions (ask IT support).

  1. Database Setup

Enter your database details (you can get them from your hosting provider):

  • Host: usually 127.0.0.1
  • Port: usually 3306
  • Database Name: the database you created (e.g. forkiva)
  • Database User: the user that can access the database (e.g. root)
  • Database Password: the user’s password

Optionally: check Install with demo data if you want sample menus and products.

Then click Save & Continue.

  1. Create Admin Account

Enter details for the first administrator:

  • Full Name (e.g. Forkiva Admin)
  • Email Address (used to log in)
  • Password and Confirm Password

Click Save & Continue.

  1. Finish 🎉

Congratulations!
The installation is complete.

Click Back to Dashboard to log in with the admin account you created.

Done! ​

After installation:

  • Log in with your admin account at:
    https://yourdomain.com
  • Start managing your restaurant with Forkiva POS.
