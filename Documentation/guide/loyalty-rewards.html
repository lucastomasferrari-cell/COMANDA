Loyalty Rewards ​

The Loyalty Rewards module allows you to create and manage the rewards that customers can redeem using their accumulated loyalty points.
These rewards can be discounts, vouchers, free items, or tier upgrades — each with its own configuration and redemption rules.

This section provides tools to:

  • Create and configure various types of loyalty rewards.
  • Assign rewards to specific loyalty programs and tiers.
  • Define reward value, redemption limits, and availability.
  • Track reward performance and customer redemptions.

Index Page ​

The Loyalty Rewards Index Page displays all available rewards with their key details such as Icon, Name, Program, Type, Points, Redeemed Count, Customers, Status, and timestamps for creation and updates.

Filters ​

You can use the following filters to refine your search:

FilterDescription
TypeFilter rewards by type — e.g., Discount, Voucher, Free Item, or Tier Upgrade.
ProgramFilter rewards by the loyalty program they belong to.
TierFilter rewards by the associated loyalty tier.
ActivationFilter rewards by activation status — Active or Inactive.
From / ToFilter rewards created within a specific date range.

Actions ​

Each reward record supports the following actions:

ActionDescription
EditOpens the edit page for updating reward details.
DeletePermanently removes the selected reward from the system.

Bulk Actions ​

Bulk ActionDescription
Delete SelectedDeletes all selected loyalty rewards.

Create Loyalty Reward Page ​

The Create Loyalty Reward Page allows you to define a new loyalty reward with specific configurations based on the selected reward type.

Attributes ​

AttributeDescription
NameThe name of the reward.
DescriptionA short description of the reward.
ProgramThe loyalty program to which this reward belongs.
TierThe tier eligible to redeem this reward.
TypeThe reward type — Discount, Voucher, Free Item, or Tier Upgrade.
ActiveToggle to activate or deactivate the reward.

Configuration (By Reward Type) ​

Depending on the selected Reward Type, specific configuration options become available:

Discount Reward ​

FieldDescription
Expires In DaysNumber of days before the reward expires after being issued.
Minimum Order TotalThe minimum order amount required to use the discount.
Maximum Order TotalThe maximum order amount eligible for the discount.

Voucher Reward ​

FieldDescription
Code PrefixPrefix used when generating the voucher code.
Expires In DaysNumber of days before the voucher expires.
Minimum Order TotalThe minimum order amount required to redeem the voucher.
Maximum Order TotalThe maximum order amount eligible for the voucher.

Free Item Reward ​

FieldDescription
ProductSelect the free product to be rewarded. The product must have an assigned SKU to appear in the list.
QuantityThe number of free items to be given when redeemed.

NOTE

Products:
To display a product in the selection list, it must have a SKU assigned.
If a product does not have a SKU, it will not appear in the product list.

Tier Upgrade Reward ​

FieldDescription
Target TierSelect the tier the customer will be upgraded to upon redemption.

Reward Value & Points ​

FieldDescription
Points CostThe number of loyalty points required to redeem the reward.

Usage & Redemption Rules ​

FieldDescription
Maximum Redemptions Per OrderThe maximum number of times the reward can be redeemed in a single order.
Usage LimitThe total number of redemptions allowed for this reward.
Per Customer LimitThe maximum number of times each customer can redeem this reward.

Conditions ​

FieldDescription
Minimum SpendThe minimum lifetime points a customer must have to qualify for this reward.
BranchesRestrict reward availability to specific branches.
Available DaysSelect the days of the week the reward is available.

Availability & Schedule ​

FieldDescription
Start AtThe date and time when the reward becomes active.
End AtThe date and time when the reward expires.

Media ​

FieldDescription
IconUpload an image representing the loyalty reward.

Edit Loyalty Reward Page ​

The Edit Loyalty Reward Page allows modifying an existing reward configuration.
It includes the same fields as the Create Page with editable parameters for program, type, and points.

Example Workflow ​

  1. Navigate to Loyalty Rewards → Create Loyalty Reward.
  2. Enter reward details and select a reward type.
  3. Fill in type-specific configuration fields.
  4. Set redemption limits and availability dates.
  5. Upload an icon and activate the reward.
  6. Click Create to save the new reward.
  7. Rewards appear in the main list.
