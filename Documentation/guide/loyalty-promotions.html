Loyalty Promotions ​

The Loyalty Promotions module allows you to create and manage promotional events that grant bonus or multiplied loyalty points to customers.
These promotions encourage engagement and increase customer spending by offering special earning opportunities under defined conditions.

This section provides tools to:

  • Create and manage loyalty point promotions.
  • Configure multiplier and bonus point campaigns.
  • Limit promotions by date, branch, or spending conditions.
  • Track program-level promotions and their performance.

Index Page ​

The Loyalty Promotions Index Page displays all created promotions with key details such as Name, Program, Type, Value, Used, Customers, Status, and timestamps for creation and updates.

Filters ​

You can use the following filters to refine your search:

FilterDescription
TypeFilter promotions by type — e.g., Multiplier, Bonus Points, Category Boost, or New Member.
ProgramFilter promotions by the associated loyalty program.
ActivationFilter promotions by activation status — Active or Inactive.
From / ToFilter promotions created within a specific date range.

Actions ​

Each promotion record supports the following actions:

ActionDescription
EditOpens the edit page for updating promotion details.
DeletePermanently removes the selected promotion.

Bulk Actions ​

Bulk ActionDescription
Delete SelectedDeletes all selected loyalty promotions.

Create Loyalty Promotion Page ​

The Create Loyalty Promotion Page allows you to define a new loyalty promotion with detailed configurations.

Attributes ​

AttributeDescription
NameThe display name of the promotion.
DescriptionA short explanation of the promotion purpose or reward.
ProgramThe loyalty program to which this promotion belongs.
TypeThe promotion type — determines how points are awarded.
ActiveToggle to activate or deactivate the promotion.

Multiplier & Bonus Points (By Type) ​

Each promotion type has its own specific configuration for defining how points are rewarded:

Multiplier ​

FieldDescription
MultiplierThe multiplier rate applied to earned points (e.g., 2X or 1.5X).

Bonus Points ​

FieldDescription
Bonus PointsThe fixed number of bonus points awarded when eligible.

Category Boost ​

FieldDescription
MultiplierThe multiplier rate applied to points earned in a specific product category.

New Member ​

FieldDescription
MultiplierThe multiplier rate for points earned by new members during their introductory period.

Conditions (By Type) ​

Each promotion type includes its own conditional rules for eligibility:

Multiplier ​

FieldDescription
Minimum SpendMinimum order amount required to qualify for this promotion.
BranchesRestrict promotion availability to specific branches.
Available DaysSelect the days of the week when the promotion is active.

Bonus Points ​

FieldDescription
Minimum SpendMinimum order amount required to qualify for this promotion.
BranchesRestrict promotion availability to specific branches.
Available DaysSelect the days of the week when the promotion is active.

Category Boost ​

FieldDescription
Minimum SpendMinimum order amount required to qualify for this promotion.
BranchesRestrict promotion availability to specific branches.
Available DaysSelect the days of the week when the promotion is active.
CategoriesRestrict the promotion to specific product categories. To display a category in the list, it must have a slug assigned. If a category does not have a slug, it will not appear in the category list.

NOTE

Categories:
To display categories in the selection list, each category must include a slug.
If a slug is missing, that category will not appear when configuring the promotion.

New Member ​

FieldDescription
Minimum SpendMinimum order amount required to qualify for this promotion.
BranchesRestrict promotion availability to specific branches.
Available DaysSelect the days of the week when the promotion is active.
Valid Days After CreationThe number of days after a new customer account is created during which the promotion remains valid.

Usage Rules ​

FieldDescription
Usage LimitThe total number of times this promotion can be applied.
Per Customer LimitThe number of times a single customer can benefit from this promotion.

Availability & Schedule ​

FieldDescription
Start AtThe date and time when the promotion becomes active.
End AtThe date and time when the promotion expires.

Edit Loyalty Promotion Page ​

The Edit Loyalty Promotion Page allows you to modify an existing promotion configuration.
It includes the same fields as the Create Page with editable type-specific configurations.

Example Workflow ​

  1. Navigate to Loyalty Promotions → Create Loyalty Promotion.
  2. Enter a name, description, and select a loyalty program.
  3. Choose the promotion type (e.g., Bonus Points, Multiplier, Category Boost).
  4. Fill in all type-specific fields and conditions.
  5. Define usage rules and availability schedule.
  6. Activate the promotion and click Create.
  7. To modify an existing promotion, click Edit, make changes, and Update.
