Payroll Components ​

The Payroll Components page defines payroll building blocks used in salary structures and employee salary profiles.

Components Index Page ​

Available Filters ​

FilterDescription
TypeFilters by component type (Earning, Deduction, Employer).
Tax Base EffectFilters by tax effect behavior.
ActiveShows active/inactive components.
FromStart date of created-at range.
ToEnd date of created-at range.

Table Columns ​

  • Code
  • Name
  • Type
  • Tax Base Effect
  • Affects Net
  • Active
  • Created At
  • Updated At

Header Actions ​

  • Create Payroll Component

Row Actions ​

  • Edit
  • Delete

System components are locked and cannot be edited/deleted.

Bulk Actions ​

  • Delete

Create/Edit Component Page ​

Form Fields ​

OptionPurpose
CodeUnique identifier for the component.
NameMulti-language component name.
TypeComponent category: Earning, Deduction, Employer.
OrderDisplay/processing order.
Tax Base EffectTax behavior of the component.
Affects NetControls whether this component affects employee net pay.
ActiveEnables/disables the component.

Tax Base Effect Options ​

OptionMeaning
Taxable EarningEarning included in taxable base.
Non-Taxable EarningEarning excluded from taxable base.
Pre-Tax DeductionDeduction applied before income tax.
Post-Tax DeductionDeduction applied after income tax.
Not ApplicableTax base effect not used (typically employer-only).

Type-to-Tax Effect Rules ​

  • Earning allows only Taxable Earning or Non-Taxable Earning.
  • Deduction allows only Pre-Tax Deduction or Post-Tax Deduction.
  • Employer allows only Not Applicable.
