Attendance Reports ​

The Attendance Reports page contains multiple report views for attendance analysis.

Available report tabs:

  • Daily Attendance
  • Monthly Summary
  • Exceptions
  • Status Summary

All tabs use Smart Data Table capabilities: sorting, pagination, filter panel, and export tools.

Daily Attendance Report ​

What this report shows

The Daily Attendance report provides one row per employee/day/shift record in the selected date range.
It is the operational report used to review day-by-day attendance outcomes and time calculations.

When to use it

  • Verify attendance for a specific day or date range
  • Check late/early/overtime values before payroll processing
  • Audit branch/shift attendance compliance

Monthly Summary Report ​

What this report shows

The Monthly Summary report aggregates attendance totals per employee for a selected month and year.
It summarizes minutes instead of listing daily rows.

When to use it

  • Review employee monthly attendance performance quickly
  • Compare worked, late, and overtime totals between employees
  • Prepare monthly HR or payroll review meetings

Exceptions Report ​

What this report shows

The Exceptions report isolates problematic or special records, such as absences, incomplete days, and auto-closed attendance.

When to use it

  • Investigate attendance gaps and anomalies
  • Follow up with managers on unresolved attendance issues
  • Track records that may require manual adjustment

Status Summary Report ​

What this report shows

The Status Summary report aggregates attendance status counts per employee over a date range, including present/absent/leave/holiday/off and late counts, plus total worked minutes.

When to use it

  • Evaluate attendance behavior trends by employee
  • Compare status distribution across teams/departments
  • Support managerial review and policy decisions

Date Defaults ​

When no date range is provided in date-based attendance reports, the system uses the current month as the default range.
