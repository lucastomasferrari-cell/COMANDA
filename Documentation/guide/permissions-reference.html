Permissions Reference ​

This document provides a complete list of system permissions available in Forkiva.
Each permission controls access to a specific feature or action within the system.

🔑 Translations ​

  • admin.translations.index – View all translations.
  • admin.translations.edit – Edit translation entries.

⚙️ Settings ​

  • admin.settings.edit – Update system settings.
  • admin.currency_rates.index – View currency exchange rates.
  • admin.currency_rates.edit – Update currency exchange rates.

🏢 Branches ​

  • admin.branches.index – View branches.
  • admin.branches.show – View branch details.
  • admin.branches.create – Create a new branch.
  • admin.branches.edit – Edit branch details.
  • admin.branches.destroy – Delete a branch.
