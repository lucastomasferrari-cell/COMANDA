Vouchers ​

The Vouchers module allows you to manage voucher-based promotions in your POS system. You can create, edit, and manage vouchers across branches with multiple configuration options such as type, value, validity period, and usage limits.

The Vouchers section provides tools to:

  • Create new voucher codes.
  • Edit or deactivate existing vouchers.
  • Filter and search vouchers by multiple criteria.
  • Manage activation status, limits, and applicable branches or products.

This module helps maintain control over voucher campaigns and ensures voucher codes are applied correctly based on predefined conditions.

Index Page ​

The Vouchers Index Page displays a list of all configured vouchers with their key details such as Name, Branch, Code, Type, Value, Usage, Activation Status, and timestamps for creation and updates.

Filters ​

You can use the following filters to refine your search:

FilterDescription
BranchFilter vouchers by branch.
TypeFilter by voucher type — e.g. Fixed or Percent.
ActivationFilter vouchers by activation status — Active or Inactive.
From / ToFilter vouchers created within a specific date range.
SearchSearch vouchers by name, code.

Actions ​

Each voucher record supports the following actions:

ActionDescription
EditOpens the edit page for updating voucher details.
DeletePermanently removes the selected voucher from the system.

Bulk Actions ​

You can perform batch operations on multiple selected records:

Bulk ActionDescription
Delete SelectedDeletes all selected vouchers from the list.

Create Voucher Page ​

The Create Voucher Page is used to define a new voucher with detailed configurations.

Attributes ​

AttributeDescription
NameThe display name of the voucher.
CodeThe unique voucher code (auto-generated or manual).
BranchSelect the branch to which the voucher applies.
DescriptionOptional text describing the voucher.
ActiveToggle to activate or deactivate the voucher.

Details ​

FieldDescription
TypeThe type of voucher (Fixed or Percent).
ValueThe voucher value (amount or percentage).
Maximum DiscountThe maximum discount that can be applied.
Minimum SpendThe minimum order amount required to apply the voucher.
Maximum SpendThe maximum order amount eligible for the voucher.

Availability ​

FieldDescription
Start DateThe date when the voucher becomes active.
End DateThe date when the voucher expires.

Conditions ​

FieldDescription
Order TypesDefine which order types are eligible (e.g., Dine-In, Takeaway).
Available DaysSelect which days of the week the voucher is available.
CategoriesRestrict the voucher to certain product categories.
ProductsRestrict the voucher to specific products.

> **Categories:**

To display the list of categories in the dropdown, each category must have a slug value assigned.
If a category does not have a slug, it will not appear in the category list.

Products:
To display the list of products in the dropdown, each product must have a SKU value assigned.
If a product does not have a SKU, it will not appear in the product list.

Limits ​

FieldDescription
Usage LimitThe total number of times the voucher can be used.
Per Customer LimitThe number of times each customer can use the voucher.

Once all fields are filled, click Create to save the new voucher.

Edit Voucher Page ​

The Edit Voucher Page allows updating an existing voucher. It includes the same fields as the Create Page:

Example Workflow ​

  1. Navigate to Vouchers → Create Voucher.
  2. Fill in all required fields including type, code, and value.
  3. Save to activate the voucher.
  4. Vouchers appear in the main list.
  5. To modify an existing voucher, click Edit, adjust configurations, and Update.
