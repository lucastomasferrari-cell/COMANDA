Products ​

The Products module allows you to manage all items available on your restaurant menus. Each product can have pricing, taxes, ingredients (recipes), media, and options for customization.

Note:

  • If the user is not assigned to a specific branch, products are displayed from the default branch active menu.
  • If the user is assigned to a specific branch, only the products for that branch’s active menu are displayed.

Index Page ​

The Products index page lists all available products in a tabular view.

Filters ​

  • Activation – Filter by Active or Inactive products.
  • From – Filter by creation start date.
  • To – Filter by creation end date.

Actions ​

  • Create Product – Add a new product.
  • Row Actions
    • Edit Product – Modify product details.
    • Delete Product – Remove a product from the system.
  • Bulk Actions
    • Delete Products – Remove multiple selected products at once.

Displayed Columns ​

  • Thumbnail
  • Name
  • Price (with special price if set)
  • Activation status
  • Created at
  • Updated at

Ingredients (Product Recipe) ​

Products can be linked to ingredients for recipe and stock tracking.

Each ingredient requires:

  • Ingredient – Select from existing ingredients (Required)
  • Quantity – Define exact quantity used (Required)
  • Loss % – Optional, specify percentage of ingredient loss (e.g., evaporation, waste)
  • Note – Optional, additional remarks

This ensures accurate inventory deduction and cost calculation when the product is sold.

Options ​

Options allow adding customization for products (e.g., Ice Level, Size, Add-ons).

Attributes: ​

  • Name – Required, translatable
  • Type – Dropdown, Radio, Checkbox, Text, etc.
  • Required – Mark option as mandatory or optional

Option Values (for multiple-choice options): ​

  • Label – Display text (translatable)
  • Price – Additional cost (optional)
  • Price Type – Fixed or Percentage
  • Ingredients – Can link ingredients per option value (e.g., “Extra Cheese” uses Cheese ingredient).

Single-value Options: ​

  • Price – Additional cost (optional)
  • Price Type – Fixed or Percentage
  • Ingredients – Can also assign ingredients.

Create Product Page ​

When creating a product, the following sections are available:

  1. General Information

    • Add Name, Description, Categories, Taxes, and Active status.

  2. Media

    • Upload a base image for the product.

  3. Pricing

    • Set base price.
    • Optionally configure special pricing and periods.

  4. Additional

    • Define “New From” and “New To” dates for promotional purposes.

  5. Ingredients

    • Add product recipe with ingredient mapping.

  6. Options

    • Define customization options with values and pricing rules.

Edit Product Page ​

Editing a product provides all fields from the Create Page with the addition of:

  • Update Ingredients – Modify, add, or remove recipe items.
  • Update Options – Adjust option groups, values, and ingredient mappings.
  • Price adjustments – Update base or special prices.
  • Status – Toggle active/inactive.

This ensures products remain accurate when menu changes occur.

Notes ​

  • Products can belong to multiple categories and be taxed with multiple tax rules.
  • Ingredients mapping is crucial for inventory tracking and profit margin analysis.
  • Special price rules help run promotions and time-limited offers.
  • Options provide flexibility for customers to customize orders while keeping stock accurate.
