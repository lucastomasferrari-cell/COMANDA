Payroll Reports ​

The Payroll Reports page provides multi-report payroll analysis with dynamic report switching.

Reports Page ​

Use the report selector action to switch between report types.

Available Report Types ​

  • Run Summary
  • Register
  • Earnings by Component
  • Deductions by Component
  • Taxable Base Summary
  • Income Tax Summary
  • Social Security Summary
  • Net Distribution
  • Exceptions
  • Variance
  • Department Cost
  • Overtime Summary
  • Absence Deductions
  • Journal Preview
  • YTD

Common Report Filters ​

Depending on report type, available filters include:

FilterDescription
BranchLimits report scope to a branch.
RunFilters by payroll run ID.
PeriodFilters by payroll period.
EmployeeFilters by employee.
DepartmentFilters by department.
YearUsed by YTD report.

Default Filter Values ​

  • Run Summary defaults: include_voided=false, include_cancelled=false
  • Net Distribution default bucket size: 300
  • YTD defaults to current year

Report Descriptions ​

Run Summary ​

Aggregated totals by run and currency, including gross, net, deductions, taxable base, and employer cost.

Register ​

One row per payslip for reconciliation and payroll audit.

Earnings by Component ​

Totals and averages by earning component and tax-base effect.

Deductions by Component ​

Totals and averages by deduction component and tax-base effect.

Taxable Base Summary ​

Taxable earnings, non-taxable earnings, pre-tax deductions, and resulting taxable base summary.

Income Tax Summary ​

Tax-policy and bracket-level taxable/tax amount summary.

Social Security Summary ​

Employee and employer social-security contribution breakdown.

Net Distribution ​

Distribution buckets of net pay values across employees.

Exceptions ​

Payslips that meet exception conditions (such as cancelled/void cases based on filter scope).

Variance ​

Compares current run metrics against previous run metrics per employee.

Department Cost ​

Payroll cost distribution by department and headcount.

Overtime Summary ​

Raw/payable overtime minutes and overtime amount by employee.

Absence Deductions ​

Impact of absent/unpaid leave days on deduction totals.

Journal Preview ​

Informational accounting-style debit/credit summary for payroll.

YTD ​

Year-to-date gross, tax, and net totals per employee.

Export Options ​

Reports support export methods from report actions:

  • CSV
  • XLS
  • XLSX
  • PDF
  • JSON
